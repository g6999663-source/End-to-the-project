## Неделя 6 – ETL pipeline full/incremental, state
- **Проблемы:**
  - `ModuleNotFoundError: No module named 'src'` – решение: `sys.path.insert` и `pipeline.bat` с `PYTHONPATH`.
  - `KeyError: 'date_range'` – из-за неправильной вложенности в YAML; исправлено вынесением секции наверх.
  - `JSONDecodeError` при пустом `state.json` – добавлена проверка размера файла.
  - Отсутствие пользователя `myuser` в PostgreSQL – замена на `postgres` в строке подключения.
  - Ошибки отступов в `load_incremental.py` – предоставлен исправленный код.
- **Результат:** единый CLI-пайплайн с full/incremental, watermark, идемпотентностью.
